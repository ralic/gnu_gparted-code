This is GParted live 0.26.1-5.
=============================

Release Notes
-------------
New in this release from 0.26.1-5:

 * The underlying GNU/Linux operating system was upgraded. 
   This release is based on the Debian Sid repository (as of 2016/Jul/23).
 * New upstream fluxbox 1.3.7+git7525226-drbl3. The window bars missing issue
   is gone.
 * Add package xserver-xorg-legacy so that no need to use workaround for
   virtualbox to enter X.
   Ref: https://bugs.debian.org/cgi-bin/bugreport.cgi?bug=807015
